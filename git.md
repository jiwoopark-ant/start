# git 명령어
- 'git init'    
-현재 폴거에 `.git` 폴더를 생성
- `git add<file, folder name>` 
    - `working  directory`에서 staging area로 추가
    - 일반적으로 모든 파일,폴더를 추가하기 위해 아래의 코드를 사용
    - `git add,`



## 설정

- `git status`  
    - 현재 상태를 체크하는 명령어 

 - `git config` 
    - git 설정울 하는 명령어
    - 일반적으로 `--global` 옵션으로 최초 한번 사용
